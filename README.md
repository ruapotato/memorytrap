Memorytrap
=================
The purpose of this tool is to allocate n GB of main memory

Dependencies
-------------------------
 * procps
 * Linux Operating system
 * make, gcc and alike

Usage
-------------------------
        ./memorytrap <Desired Used in GB>


Building
-------------------------
	git clone (This URL)
	cd memorytrap
	make
